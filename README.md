# Places We Keep

个人旅行档案网页。当前第一篇是 `2026年 · 7月 · 成都`，包含每日路线、互动地图、地点详情、吃喝提示和照片记录入口。

## 打开方式

直接在浏览器里打开：

```text
/Users/qiaqia/Documents/旅行/my-travel-journal/index.html
```

相册页：

```text
/Users/qiaqia/Documents/旅行/my-travel-journal/album.html
```

## 当前内容

- 顶部导航：时间、地点、相册
- 首页标题：2026年 · 7月 · 成都
- 7月29日：青城山前山 + 都江堰 + 南桥 / 灌县古城自驾线
- 地图地点：太古里、成都博物馆、人民公园、宽窄巷子、熊猫基地、建设路、青城山、都江堰、南桥、武侯祠、玉林、九眼桥
- 交互：切换日期、点击地图点位、加入今日路线、相册入口

## 替换照片

把真实旅行照片替换到 `assets` 目录里的同名文件即可：

- `hero-chengdu.png`
- `qingcheng-dujiangyan.png`
- `peoples-park-tea.png`
- `chengdu-night.png`

后续可以继续增加照片，再扩展 `album.html` 里的相册栏目。
